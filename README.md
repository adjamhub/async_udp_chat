# Async UDP Chat

Just an easy and didactic project to implement an async client-server chat based on the UDP sockets


## server mode

Run the server calling

$ async_udp_chat --server


## client mode (textual)

Run the client calling

$ async_udp_chat --client


## client mode (wx GUI)

If you prefer, you can run the client in wx GUI mode.

$ async_udp_chat --gui
